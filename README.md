### What I learned

It was a good first challenge, the main problem I had was to keep the footer at the bottom of the screen!

that little negative margin-bottom is magic!

```css
body {
    margin: 0;
    background-color: hsl(212, 45%, 89%);
    margin-bottom: -5em;
}
```
